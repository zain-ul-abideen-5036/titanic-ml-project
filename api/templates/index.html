About the Dataset
RMS Titanic — April 15, 1912 Of 2,224 passengers and crew, only 710 survived (32%). The ship carried lifeboats for only 1,178 people. The rule of "women and children first" was strictly enforced on the Titanic, which heavily influenced survival outcomes across passenger classes.
This model was trained on 891 passenger records from the Kaggle Titanic dataset using RandomForestClassifier with GridSearchCV hyperparameter tuning.
